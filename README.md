# Smart Complaint Analyzer (Advanced)

## Problem
Urban complaints are not well organized, making it difficult to manage civic issues.

## Solution
A Java-based system that:
- Categorizes complaints
- Assigns priority
- Stores data in files
- Allows searching

## Features
- Add complaint
- Auto categorization
- Priority detection
- File storage
- Search functionality

## How to Run
javac Main.java
java Main

## Technologies
- Java
- OOP
- File Handling
